Projetos Desenvolvidos

Meus projetos mais recentes desenvolvidos para o MTA (Multi Theft Auto)

  • Corridas Ilegais

    O sistema tem como principal objetivo corridas ilegais, que são iniciadas pela administração em eventos. Tem um sistema de ranqueamento conforme a quantidade de vitórias de cada jogador que aparecerá no início das corridas.

  • Empregos Ilegais

    Empregos do ramo ilegal, alguns conhecidos pela comunidade, outros novidade. Foi uma tentativa de fazer um pouco diferente dentro do servidor com trabalhos mais interativos.

    Mostrar Vídeo

  • Login/Registro Completo

    Esse sistema foi uma encomenda, se trata de um sistema de login completo. Cadastro de contas e criação de personagens, punição no personagem, alteração de senha, excluir personagem, moedas por conta (compartilhada com todos os personagens). Utilizei SQLite para o bancos de dados.

    Acessar Figma

  • Gerenciamento de Mecânicas

    Também foi uma encomenda, se trata de um painel de gerenciamento de mecânica pela administração. O administrador do servidor pode criar e editar todas as informações após a criação e consegue criar pontos adicionais que estão associadas a uma determinada mecânica, podendo ser acessada de outros lugares. Utilizei SQLite para o banco de dados.

    Mostrar Vídeo

  • Plotagem de Viaturas

    Sistema simples que permite alterar texturas de um veículo através de shader, tendo as permissões necessárias, o jogador consegue selecionar a plotagem de acordo com o modelo da viatura. Possui um gerenciamento interno das imagens pra não ter sobrecarga.

    Mostrar Vídeo

  • Ônibus Interativo

    Esse sistema permite interativamente selecionar o lugar onde o jogador gostaria de ir. Após selecionar o destino, o jogador era movido para um ônibus com música ao vivo de uma rádio FM.
